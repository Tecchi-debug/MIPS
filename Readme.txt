How to run tinyvm.c 

Step 1: Compile the program

gcc -o runme tinyvm.c

This will make a executable we can run named 'runme'

Step 2: run with elfMarie.txt

./runme elfMarie.txt

this will run the 2 x 3 marie program

make sure to input 2 and 3 in order.
